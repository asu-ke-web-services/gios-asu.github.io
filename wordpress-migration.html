Wordpress Theme

Background

The ASU Webstandards Wordpress Theme has been tested with wordpress (single and multisite) versions 3.5 - 4.0+.

Authentication

It is highly recommended to use CAS authentication for all wordpress instalations. This prevents allot of attacks and other exploits. One popular plugin for doing CAS authentication is CAS Maestro. Configuration is very simple, ASU's CAS server is at weblogin.asu.edu.

Enabling auto updates from Github

We recommend using the github-updater plugin, despite it not being sanctioned by the wordpress.org repository standards (read why). This plugin will integrate into wordpress's auto updating hooks and poll github for new tagged releases for themes and plugins.

Note The Github Updater is a forked repo from this github-updater repo in order to implement a bug fix for SSH2 connections. Once Wordpress version 4.1 comes out, the original repo will be the plugin to use.
