Getting Started

Download

The ASU Web Standards Bootstrap Addon is available threw Github. The raw SCSS files and the current CSS build are available in the repository.

Download Bootstrap

Using the Web Standards

The Web Standards are built on top of Bootstrap and require the Google Font "Roboto". The following is an example of what your head tag might look like: 

          
<link href='//fonts.googleapis.com/css?family=Roboto:900,400,300,700,500' rel='stylesheet' type='text/css' /><!-- The main app css file -->
<link href="./css/bootstrap.css" rel="stylesheet" type="text/css" />
<link href="./css/bootstrap-asu.css" rel="stylesheet" type="text/css" />
<link href="./css/bootstrap-asu-theme-base.css" rel="stylesheet" type="text/css" />

bootstrap.css would be the latest version of Twitter's Bootstrap.

bootstrap-asu.css is the latest build from the GitHub Repo of the ASU Web Standards.

bootstrap-asu-theme-base.css is an initial set of CSS that themese the your website to the default ASU theme. This file can be customized by editing the SCSS files that are found /scss/custom in the GitHub repo and compiling them.
