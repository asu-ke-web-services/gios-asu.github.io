Welcome to the ASU Web Standards Bootstrap Project

The new web standards that are being rolled out to the schools at ASU are built upon responsive design, common widgets and themes, and Twitter's Bootstrap.

This Bootstrap project is a complete addon to Bootstrap. Everything available in Twitter's Bootstrap is also available when using this addon. Checkout the Bootstrap website for more details.
