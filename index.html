Welcome to the ASU Web Standards Bootstrap implementation

This site is intended to be a guide for the new ASU Webstandards Bootstrap implementation and software projects that use the webstandards boostrap theme.

The new web standards that are being rolled out to the schools at ASU are built upon responsive design, common widgets and themes, and Twitter's Bootstrap.

The ASU Web Standards Addon is used in conjunction with Bootstrap. Everything available in Twitter's Bootstrap is also available when using this addon. Checkout the Bootstrap website for more details on what default features are available.
